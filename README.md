## ğŸ“¦ Installation

1. Clone the repository:
```bash
git clone https://github.com/petsereypanha/data-engineer.git
cd data-engineer
```

2. Create a virtual environment (recommended):
```bash
python -m venv venv

```

3. Install required packages:
```bash
pip install -r requirements.txt
```
